We ask participants to upload a zip file that contains the following 2 files:

  • discovery.csv: This should contain results on COCO 2014 train split in the following format - < image_id >, <x1>,<y1>,<x2>,<y2>,<cluster_id>
  • detection.csv: This should contain results on COCO minival in the following format - <image_id>, <x1>,<y1>,<x2>,<y2>,<cluster_id>,<conf_score>

    • Here <image_id> is the unique identifier of an image as used in COCO 2014 train set.

    We further include code in our github repo that generates these csv files and evaluates the results.
